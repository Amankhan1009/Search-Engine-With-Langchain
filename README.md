# Search Engine With Langchain

A simple search engine application built using Python, Streamlit, and LangChain.

## Features

- Uses LangChain for building a retrieval-based search.
- Integrates streaming UI with Streamlit.
- Supports embeddings + vector store for semantic search.

## Installation

```bash
git clone https://github.com/Amankhan1009/Search-Engine-With-Langchain.git
cd Search-Engine-With-Langchain
pip install -r requirements.txt
```
## Usage
# Run the application:
```bash
streamlit run app.py
```

## Files

- app.py — Main application entry point.
- requirements.txt — Dependencies.
